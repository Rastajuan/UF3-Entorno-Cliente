UF3 ENTORNO CLIENTE

Actividad 1

Enunciado

a) Crear un formulario de contacto donde se pida el nombre, horario por la mañana o tarde (sólo se podrá escoger una de las dos opciones) y una casilla para aceptar las normas del registro. El formulario tendrá que tener un botón que al hacer clic se tendrá que validar el formulario. Para validar el formulario se tendrá que comprobar que la casilla del nombre no esté vacía, que se ha seleccionado un horario y se han aceptado las normas de registro. Se indicará con un Alert cuál de estas condiciones no se cumplen.

Actividad 2

Enunciado

Cada vez que se haga clic encima lo botón se tiene que crear un nuevo elemento de tipo párrafo ("p") con el texto “¡Hola Mundo!”. Los nuevos elementos tienen que aparecer dentro de la capa "div id="contenido" "" ( como hijos de esta, dentro de lo estructura DOM). Añadir además 3 botones que permitan alinear el contenido de la capa div id="contenido" a la izquierda, a la derecha o al centro. Para hacerlo tendrás que añadir el atributo “align” con los diferentes valores left, center o right.
